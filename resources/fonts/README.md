# Fontes do Sistema

Coloque aqui os arquivos `.ttf` das fontes selecionáveis no painel administrativo.
Isso garante que o servidor consiga gerar as artes (artilhagem, MVP, etc) com a identidade visual correta do clube.

## Estrutura recomendada:
- `Inter-Regular.ttf`
- `Inter-Bold.ttf`
- `Montserrat-Bold.ttf`
- `Roboto-Regular.ttf`

## Backup
Se uma fonte não for encontrada nesta pasta, o sistema utilizará o padrão `Inter-Regular.ttf` ou a fonte genérica do sistema.
